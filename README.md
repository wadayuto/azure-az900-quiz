# AZ-900 過去問道場

Microsoft Azure Fundamentals (AZ-900) の学習用 Web アプリ（過去問演習）。

バックエンド不要の純クライアントサイド SPA（HTML + CSS + バニラ JS）。学習の進捗はブラウザの LocalStorage に保存されます。

## 機能

- **問題演習**: カテゴリ／問題数／出題順（シャッフル・番号順・苦手優先）を選んで出題。即時採点＋解説。
- **問題一覧**: カテゴリ別に全問を閲覧、詳細モーダル。
- **Azure サービス一覧**: 主要サービスをカテゴリ別のアコーディオンで表示。カードをタップで初学者向けのやさしい解説がポップアップ。
- **解説内のサービス連携**: 解説に登場したサービス名をタップすると、同じ解説ポップアップが開く（一覧と同一データで同期）。
- **マイページ**: カテゴリ別正答率、学習カレンダー。

## ローカルでの起動

```bash
cd webapp
python3 -m http.server 9901
# http://localhost:9901 を開く
```

`file://` で直接開いても動作します。

## ディレクトリ構成

```
webapp/
├── index.html
├── style.css
├── questions.js      # 問題データ (window.QUESTIONS)
├── services.js       # サービスデータ (window.SERVICES)
├── img/              # サービス解説用の画像
└── js/
    ├── app.js        # メインロジック
    └── storage.js    # LocalStorage 層
```
